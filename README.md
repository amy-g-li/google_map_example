Getting Started
---------------
Before developing, make sure you are in the correct virtual environment:

    $ cd ~/Projects/map/
    $ . .venv/bin/activate

Start your server
-----------------

    $ python3 -m flask --debug --app map run

Navigate to: http://127.0.0.1:5000/map
